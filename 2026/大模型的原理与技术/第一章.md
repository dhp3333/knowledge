# 1.1 基于统计的语言模型

n-grams
在马尔可夫假设和极大似然的思想下，设计基于相对频率的模型。使得该模型可以最大概率的生成语料库中的语料。

# 1.2 基于学习的语言模型

统计侧重于设计一个模型描摹已知，学习侧重于找到一个模型描摹未知。
基于学习的模型可以更好的应对语言的复杂性。设定合理的学习目标，利用优化方法找到一组尽量符合学习目标的神经网络参数。并在未知的验证集上进行验证，遴选。

学习的要素：经验知识、学习载体、常识、学习方法、学习目标。

机器学习的要素：训练数据、假设类、归纳偏置、学习算法、学习范式。

1. 训练数据 Training Data 训练数据的数量和质量（多样性、噪声等因素）都会影响机器学习性能。
2. 假设类（Hypothesis Class ，H）所有可能机器学习模型的集合，其中的元素为假设。实践中，通常指候选模型的参数空间为假设类。当前大语言模型用神经网络作为假设空间。
3. 归纳偏执限制对某些假设进行选择。对于语言模型而言，常用的归纳偏执是上下文间存在关联。CNN RNN Transformer
4. 学习范式 监督学习、无监督学习、强化学习。
5. 学习目标 不同学习范式具有不同的学习目标。经验风险最小化是最为常见的目标之一，最小化模型在训练集上的错误。
6. 损失函数 Loss function
7. 学习算法对损失进行优化。最优化理论。
8. 泛化误差 机器学习的目的在于减少泛化误差（Generatialization Error），即真实误差
